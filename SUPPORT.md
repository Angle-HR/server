## Asking a Good Question

The better the question, the faster we can help. Please include:

- **What you're trying to do** — describe the goal, not just the error.
- **What you've already tried** — this avoids suggestions you've already ruled out.
- **Your environment** — OS, runtime/language version, project version.
- **A minimal example** — a small, self-contained snippet that reproduces the issue is worth a thousand words.
- **Relevant logs or errors** — paste the actual output, not a summary of it.

---

## What This Repo Is Not

To keep issues and discussions useful for everyone, please avoid using them for:

- General programming questions unrelated to this project.
- Debugging your own application code (unless a project bug is suspected).
- Requests for paid consulting or private support.
